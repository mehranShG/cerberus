## Description

User authentication API with [Nest](https://github.com/nestjs/nest) framework

## Installation

```bash
$ npm install
```

remove example from .env example to access .env files

## Running the app

```bash
# development
$ npm run start
# watch mode
$ npm run start:dev
# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
# e2e tests
$ npm run test:e2e
# test coverage
$ npm run test:cov
```
